### Question

You are given a string of lowercase letters.

You are allowed to delete adjacent pairs of lowercase letters, as many times as you want

Delete as many characters as possible and print the resulting string.

Example:
```
aaabccddd
abccddd
abccd
abd    <-- final answer
```
