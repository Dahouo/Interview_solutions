### Question

Given two integers `l` and `r`, find the maximal value of `a xor b`, written `a ⊕ b`, where `a` and `b` and  satisfy `l <= a <= b <= r`

For example, if `l = 11` and `r = 12`, then

```
11 ⊕ 11 = 0
11 ⊕ 12 = 7
12 ⊕ 12 = 0
```

Our maximum value is 7.
